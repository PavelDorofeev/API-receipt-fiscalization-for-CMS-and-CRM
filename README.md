Рады представить АПИ БИТ драйвер ККТ разработчикам облачных решений (для  разных CMS,CRM).
АПИ решает проблему с фискализацие чеков на локально подключенном к компьютеру оборудовании, а точнее
на кассовых аппаратах (РФ, ФЗ54) моделей Атол и Меркурий.

Чтобы понять как работает АПИ перейдите на страницу https//kkmspb.ru/api/test.php, укажите несколько настроечных параметров и проверьте фискализацию чека.

Чтобы реализовать механизм взаимодействия  используйте АПИ протокол обмена.

Скачать тестовую страницу test.php можно здесь на гитхабе.
