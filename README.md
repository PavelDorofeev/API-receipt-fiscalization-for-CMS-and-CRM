Рады представить АПИ БИТ драйвер ККТ разработчикам облачных решений (для  разных CMS,CRM).
АПИ решает проблему с фискализацие чеков на локально подключенном к компьютеру оборудовании, а точнее
на кассовых аппаратах (в РФ, ФЗ54) моделей Атол и Меркурий.

Чтобы понять как работает АПИ скачайте страницу test.php, разместите у себя на сайте, откройте 
и передайте задание для пробития чека (в формате json) на сайт kkmspb.ru.
